EOS M Scout v1.0.0

Free APK for EOS M framing/scouting on your phone.

Features:
- Live EOS M red frame preview with lens (16–135mm incl. 55mm)
- Magic Lantern crops (Default, 5.2K, 5K FRTP, 4K 16:9, etc.)
- Speed Booster toggle
- Grids (thirds, golden, horizon, etc.)
- Social crops (4:5, 1:1, 9:16)
- SCOUT button saves screenshot with overlays + metadata
